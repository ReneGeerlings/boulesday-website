# boulesday-website

Marketing- en support-pagina's voor BoulesDay (iOS-app voor wedstrijdleiders bij pétanque-toernooien).

## Pagina's

- `index.html` — landing met korte intro en TestFlight-aanmelding
- `privacy.html` — privacyverklaring (verplicht voor App Store + TestFlight)
- `support.html` — FAQ en contact voor testers en gebruikers

## Lokaal bekijken

Open `index.html` direct in een browser, of start een lokale server:

```sh
cd website
python3 -m http.server 8080
```

Daarna open `http://localhost:8080` in je browser.
